# Dashboard

## Introduction
A web application that visualizes the 5 different sorting algorithms: BubbleSort, InsertionSort, SelectionSort, MergeSort. It is built using React and Bootstrap as a CSS framework.

1. VS Code
2. yarn 

## Cloning Respository
```
git clone git@github.com:Rixant/Rixant/Sorting-Algorithms-Visualization.git
```

## Installation

```
cd Sorting-Algorithms-Visualization
yarn install
yarn start
```
The web app starts running at: https://localhost:3000
